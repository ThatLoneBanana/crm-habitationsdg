# UI kit — CRM Habitations DG

Full interactive, high-fidelity recreation of the Habitations DG CRM, composed from the design-system components and brand tokens.

## Run
Open `index.html`. It loads React + Babel, the Tabler icon webfont, Leaflet (for the map), `../../styles.css`, the compiled `../../_ds_bundle.js`, the data layer, and every screen file. Use the **sidebar** to move between screens. (The bundle is generated by the compiler — if you just edited the system, it appears after the turn completes.)

## Screens
| Nav | File | Notes |
|---|---|---|
| Dashboard | `Dashboard.jsx` | Nicolas's morning view — metrics, alerts, week agenda, project rail. |
| Projets | `ProjetsList.jsx` | Dense table; phase filter chips, search, sort, row actions. |
| _(Nouveau projet)_ | `ProjetCreate.jsx` | 4-step flow Client → Projet → Cédule → Confirmation, with **« passer la cédule »**. |
| _(open a project)_ | `ProjectPage.jsx` | Header, overview, tabs: Cédule · Extras · Paiements · GCR · Costing · Documents. |
| Cédule (tab) | `GanttView.jsx` · `CeduleEditor.jsx` | Gantt (consultation) + editor (édition). |
| Costing (tab) | `Costing.jsx` → `CostingTab` | Financial summary, engaged vs budget, dépenses par catégorie + main-d'œuvre. |
| GCR (tab) | `GcrTab.jsx` | Checklist, 3 inspections, activity journal. |
| Costing | `Costing.jsx` → `CostingGlobal` | All projects: revenus / dépenses / profit / marge with health color. |
| Feuilles de temps | `FeuillesTemps.jsx` | Consultation · Saisie (weekly grid, hours vs 36,5 h max) · Employés. |
| Clients / Fournisseurs | `ClientsFournisseurs.jsx` | Cartes / liste toggle. |
| Carte | `Carte.jsx` | Leaflet, markers by phase/type, interactive legend-filter. |
| Paramètres | `Parametres.jsx` | Général · Mon compte · Utilisateurs · Journal d'activité. |
| Vue mobile | `MobileViews.jsx` | Dashboard + page projet adapted to a narrow screen. |

## Data
- `data.js` — `window.DG`: Québec money/date formatting, working-day cascade, auto task-status, the 43-step schedule, seed projects.
- `data-ext.js` — extends `window.DG`: clients, fournisseurs, employés, costing (per-project + global, health by margin), timesheets, GCR, geo-coordinates, utilisateurs, activity journal.

## Fidelity notes
- Schedule, status colors, cascade and conflict logic mirror the production `src/lib/*`. The visual layer is the elevated direction.
- Showcase project is Éric Lemieux · 210 Rang Saint-Charles (CHANTIER, ~71%) for a rich Gantt status spread; the public client view uses Michel Rodrigue · 18 Rue des Érables (LIVRAISON).
