Small status / category label with a tint + ink color pair.

```jsx
<Badge tone="success" dot>Payé</Badge>
<Badge tone="warning">Non signé</Badge>
<Badge tone="danger" pill>En retard</Badge>
```

Tones: `neutral` · `accent` (red) · `success` · `warning` · `danger` · `info` · `outline`. `pill` for fully-rounded, `dot` for a leading status dot. For the five project lifecycle phases, use **PhaseBadge** (it owns the phase palette).
