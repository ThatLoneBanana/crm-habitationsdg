Square icon-only button for toolbars and row affordances. Always pass a `label` (used as `aria-label` + tooltip).

```jsx
<IconButton label="Plus d'options"><i className="ti ti-dots" /></IconButton>
<IconButton variant="outline" label="Imprimer"><i className="ti ti-printer" /></IconButton>
<IconButton variant="danger" label="Supprimer"><i className="ti ti-trash" /></IconButton>
```

Variants: `ghost` (default), `outline`, `danger`. Sizes: `sm` 24 · `md` 28 · `lg` 32.
